    linux的桌面图形架构图，使用client与server模式，server使用x11（X Window System protocol）协议，client通过x11协议告诉server怎么显示，显示什么，就像http协议，xlib是封装协议的c库，使用xlib可以方便与xserver建立连接，显示图像，不用关注协议具体的字节格式等等

    xlib是对x11显示服务协议的封装

    x11定义了客户端到服务器端的显示模式
